# 🏦 银行股双擎量化系统 (Bank Quant System)

基于 Cloudflare (Pages + Worker) 构建的 Serverless A股银行股专属量化投研终端。

## ✨ 核心特性
* **0 成本部署**：全套架构依托 Cloudflare 免费额度，无需购买服务器。
* **五维长线打分模型**：融合 5 年均值 ROE、股息率、估值分位、不良率、拨备率进行量化评分。
* **极速并发爬虫**：直连腾讯骨干网 0.05 秒刷新现价，纯 JS 并发抓取东方财富最新财报。
* **AI 策略军师**：接入大模型流式输出 (SSE)，深度解读财报并结合宏观经济给出操作评级。
* **历史回测引擎**：基于 Tushare 历史数据，秒级回测买卖防线，计算年化收益与最大回撤。

## 🛠️ 部署指南
1.  **前端部署**：将 `frontend` 文件夹上传至 Cloudflare Pages。
2.  **后端部署**：在 Cloudflare 创建 Worker，贴入 `backend/worker.js` 代码。
3.  **配置 KV 空间**：绑定名为 `BANK_DB` 的 KV 命名空间用于存储用户配置与缓存。
4.  **接口打通**：在前端 `script.js` 第一行修改 `API_BASE` 为你的 Worker 域名。
5.  # 🔑 核心接口申请指南 (小白必看)

本量化系统依赖两个外部免费 API 服务：**Tushare**（用于获取海量金融数据）和 **Server酱**（用于微信消息推送）。请按照以下步骤免费申请属于你自己的密钥。

## 一、申请 Tushare 免费金融数据 Token

Tushare 是国内目前最稳定、好用的免费金融量化接口。我们的系统通过它来拉取银行股的长线历史估值和历年财报。

**操作步骤：**
1. **注册账号**：访问 [Tushare 官方网站](https://tushare.pro/)，点击右上角的“注册”。
2. **进入个人中心**：注册并登录成功后，点击右上角的个人头像，进入 **【个人主页】**。
3. **获取 Token**：在左侧菜单栏找到 **【接口 TOKEN】** 选项。
4. **复制密钥**：你会看到一串很长的英文字母和数字组合，点击右侧的 **【复制】** 按钮。
5. **填入系统**：回到我们的“银行估值看板”网页，点击右上角的 **【🔑 TS Token】** 按钮，将这串代码粘贴进去并保存即可。

> **⚠️ 注意事项 (关于积分)**：
> Tushare 的基础数据（如每日行情、财务指标）是免费的，但有一定的频率限制。本系统已内置了“智能断点续传”和“防滥用”机制，如果在首次同步时遇到进度卡住（部分银行显示“待同步”），请耐心等待 1~2 分钟后再次点击网页上的【🔑 TS Token】按钮，系统会自动补齐剩余数据。

## 二、申请 Server酱 推送 SendKey (可选)

如果你希望系统在某只银行股跌破你的“买入防线”时，能全自动发微信消息提醒你，就需要配置 Server酱。如果你不需要微信提醒，此步可跳过。

**操作步骤：**
1. **登录网站**：访问 [Server酱 官网 (SCT版)](https://sct.ftqq.com/)。
2. **微信扫码登录**：点击右上角的“登入”，推荐直接使用微信扫码登录。
3. **获取 SendKey**：登录后，在顶部导航栏点击 **【SendKey】**。这里会显示一串类似 `SCT...` 开头的密钥，点击 **【复制】**。
4. **配置微信接收通道**：
   * 点击顶部导航栏的 **【消息通道】**。
   * 选择 **【方糖服务号】** 或 **【测试号】**，按照屏幕上的提示，用微信扫码关注官方公众号。这是为了让公众号能给你发消息。
5. **填入系统**：回到“银行估值看板”网页，点击右上角黄色的 **【🔔 Push】** 按钮，将刚才复制的 `SCT...` 密钥粘贴进去并保存。

> **💡 测试推送**：
> 填写完成后，你可以在 Server酱的网页后台，点击“SendKey”页面里的“发送消息”进行一次测试，看看你的微信是否能成功收到提醒。
